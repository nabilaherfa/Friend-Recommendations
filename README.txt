1. Algoritma yang diimplementasikan
   Program ini menggunakan algoritma DFS dan BFS untuk melakukan
    * Friend Recommendation
    * Explore Friends
   Representasi graf pada program:
    * Node merepresentasikan akun
    * Sisi merepresentasikan jalur pertemanan antar akun
   Struktur data yang digunakan:
    * graph yang merupakan matriks n_account x n_account yang berisi boolean 
    * map yang merupakan list of string berisi nama-nama akun
    * n_account bertipe integer yang berisi jumlah akun pada graf
    * words yang mengubah file eksternal menjadi bentuk array of string

2. Requirement
   1. Visual Studio .NET
   2. MSAGL (https://github.com/microsoft/automatic-graph-layout)
      (https://docs.google.com/document/d/1XhFSpHU028Gaf7YxkmdbluLkQgVl3MY6gt1tPL30LA/edit?usp=sharing)

3. Cara Menggunakan Program
   1. Download folder Tubes2_13519013.zip lalu ekstrak folder tersebut
   2. Buka Visual Studio .NET
   3. Pilih Open a project or Solution
   4. Pilih file WinFormsApp3.sln pada folder src
      (Tubes2_13519013 → src → WinFormsApp3.sln)
   5. Pada solution explorer, buka file Form1.cs dengan melakukan double-click
   6. Run Form1.cs 
   7. Form akan muncul
